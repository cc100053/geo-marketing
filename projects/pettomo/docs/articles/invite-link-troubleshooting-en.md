# What to do when a PetTomo invite link does not open

PetTomo shared rooms are private invite-based spaces where members raise the same virtual pet, chat, feed with photos, decorate the room, and collect memories together. Local 1.2.0 in-app What's New copy says invite link sharing became more reliable, while the PetTomo knowledge base confirms that invite codes remain an important fallback.

This guide explains the practical checks to try when a PetTomo room invite link does not open as expected.

## Key takeaways

- A PetTomo room is joined through an invite link or invite code.
- Invite links are convenient, but device, browser, sign-in, or app-opening behavior can affect them.
- If the link does not open PetTomo correctly, manually entering the invite code is the safest fallback.
- Ask the room owner to re-share the invite if the code was copied incorrectly or regenerated.
- PetTomo invites are for private shared rooms, not public social profiles.

## First confirm the invite source

Before troubleshooting the device, confirm that the invite came from the correct PetTomo room member. A shared-room invite should be sent by someone who already has access to the room and wants you to join that private space.

If you belong to more than one room, also confirm which room the invite is for. PetTomo room data, chat, pet state, memories, and decorations are room-specific, so joining the correct room matters.

## Check sign-in and profile setup

PetTomo invite flows normally require the invited person to open the app, sign in, and complete the required profile setup before joining the room. If the invite link opens the app but the room does not appear, finish sign-in first and then try the invite again.

If the app was newly installed from the store, return to the original invite link or ask the sender for the invite code. A store install step can interrupt the first app-opening attempt.

## Use the invite code fallback

PetTomo's knowledge base explicitly treats invite codes as a fallback when links do not open the app. If the invite link fails, copy the code carefully and enter it manually inside PetTomo.

This helps when:

- The link opens in the wrong browser.
- The device does not hand the link to the app.
- The recipient installs PetTomo after tapping the invite.
- The message app trims or previews the link in an unexpected way.

## Ask the sender to re-share

If manual entry fails, ask the room owner or sender to create or share the invite again. The code may have been copied with missing characters, regenerated, invalidated, or sent for a different room.

Keep the wording cautious: an invite problem does not mean the room or pet is public, lost, or permanently unavailable. It usually means the join flow needs the right link, code, account, and app state.

## FAQ

### Can I join a PetTomo room without an invite?

PetTomo is centered on private shared rooms. You need an invite link or invite code from a room member to join a room.

### What should I try first if the link fails?

Open PetTomo, sign in, and manually enter the invite code. This is the clearest fallback when link opening is unreliable.

### Are PetTomo invite links public profiles?

No. Invite links are used to join private shared rooms. PetTomo should not be described as a public social network.

## Summary

When a PetTomo invite link does not open, treat the invite code as the reliable fallback. Confirm sign-in, check that the code was copied correctly, and ask the room owner to re-share the invite if needed.
