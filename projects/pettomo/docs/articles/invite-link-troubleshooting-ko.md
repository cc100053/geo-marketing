# PetTomo 초대 링크가 열리지 않을 때 확인할 것

PetTomo 공유 방은 초대받은 멤버가 같은 가상 펫을 함께 돌보는 비공개 공간입니다. 로컬 1.2.0 앱 내 What's New 문구는 초대 링크 공유가 더 안정적으로 개선되었다고 안내하며, PetTomo 지식 베이스는 초대 코드가 링크 문제의 중요한 대체 수단이라고 설명합니다.

이 글에서는 PetTomo 초대 링크가 열리지 않거나 공유 방에 들어가지 못할 때 확인할 점을 정리합니다.

## 핵심 요약

- PetTomo 공유 방은 초대 링크 또는 초대 코드로 참여합니다.
- 초대 링크는 편리하지만 기기, 브라우저, 로그인 상태, 앱 실행 방식의 영향을 받을 수 있습니다.
- 링크가 PetTomo를 제대로 열지 못하면 초대 코드를 직접 입력하는 것이 현실적인 대체 방법입니다.
- 코드가 잘못 복사되었거나 다시 생성되었을 수 있다면 방 멤버에게 다시 공유해 달라고 요청합니다.
- PetTomo 초대는 비공개 공유 방을 위한 것이며 공개 소셜 프로필이 아닙니다.

## 먼저 초대 출처를 확인하기

기기 문제를 확인하기 전에 초대가 올바른 PetTomo 방 멤버에게서 온 것인지 확인하세요. 공유 방 초대는 이미 방에 있는 멤버가 상대를 그 비공개 공간으로 초대하기 위해 사용하는 흐름입니다.

여러 방에 참여하고 있다면 어떤 방의 초대인지도 확인해야 합니다. PetTomo의 채팅, 펫 상태, 추억, 방 꾸미기는 방마다 분리되어 있습니다.

## 로그인과 프로필 설정 확인하기

PetTomo 초대 흐름은 보통 초대받은 사람이 앱을 열고 로그인한 뒤 필요한 프로필 설정을 마쳐야 공유 방에 참여할 수 있습니다. 링크로 앱은 열리지만 방이 보이지 않는다면 먼저 로그인을 완료하고 초대를 다시 시도하세요.

초대 링크를 누른 뒤 PetTomo를 설치했다면 첫 앱 열림이 중간에 끊길 수 있습니다. 설치 후 원래 초대 링크로 돌아가거나 상대에게 초대 코드를 다시 받아 확인하는 것이 좋습니다.

## 초대 코드를 직접 입력하기

PetTomo 지식 베이스는 초대 링크가 앱을 올바르게 열지 못할 때 초대 코드 직접 입력을 대체 방법으로 설명합니다. 링크가 잘 작동하지 않으면 코드를 정확히 복사해 PetTomo 안에서 입력하세요.

이 방법은 다음 상황에서 도움이 됩니다.

- 링크가 다른 브라우저에서 열립니다.
- 기기가 링크를 앱으로 넘기지 못합니다.
- 초대를 누른 뒤 PetTomo를 설치했습니다.
- 메신저 앱이 링크를 줄이거나 미리보기로 바꿉니다.

## 보낸 사람에게 다시 공유받기

직접 입력해도 참여할 수 없다면 방 소유자나 보낸 사람에게 초대를 다시 만들거나 다시 공유해 달라고 요청하세요. 초대 코드가 일부 빠졌거나, 다시 생성되었거나, 무효화되었거나, 다른 방의 초대였을 수 있습니다.

초대 문제는 방이나 펫이 공개되었거나 사라졌거나 영구적으로 참여할 수 없다는 뜻이 아닙니다. 대부분은 올바른 링크, 코드, 계정 상태, 앱 상태를 맞추면 해결하기 쉽습니다.

## FAQ

### 초대 없이 PetTomo 방에 참여할 수 있나요?

PetTomo는 비공개 공유 방을 중심으로 합니다. 방에 참여하려면 방 멤버가 보낸 초대 링크 또는 초대 코드가 필요합니다.

### 초대 링크가 열리지 않으면 무엇을 먼저 해야 하나요?

PetTomo를 열고 로그인한 다음 초대 코드를 직접 입력하세요. 링크 열림이 불안정할 때 가장 분명한 대체 방법입니다.

### PetTomo 초대 링크는 공개 프로필인가요?

아니요. 초대 링크는 비공개 공유 방에 참여하기 위한 것입니다. PetTomo를 공개 소셜 네트워크로 설명해서는 안 됩니다.

## 정리

PetTomo 초대 링크가 열리지 않을 때는 초대 코드를 신뢰할 수 있는 대체 수단으로 사용하는 것이 현실적입니다. 로그인 상태, 코드 복사, 초대받은 방을 확인하고 필요하면 방 멤버에게 다시 공유받으세요.
